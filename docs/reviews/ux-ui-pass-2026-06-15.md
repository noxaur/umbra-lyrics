## Top issues found

1. **Active lyric 3D scale caused horizontal bleed on mobile** — measured 394px wide active button in 375px viewport before fix.
2. **No `overflow-x-hidden` on lyrics stage** — scaled transforms could paint outside scrollport.
3. **Long lyric strings lacked wrap constraints** — JP/EN unsynced lines could clip on narrow columns.
4. **Lyrics column missing `min-w-0`** — flex child could resist shrinking in split layout at lg.
5. **Mobile header title** — track name could crowd badges without full-width truncation.

## Fixes applied

- `lyrics-stage.tsx` — `overflow-x-hidden`, `scroll-py-8`, safe-area bottom padding, inner `overflow-x-hidden`, slightly looser line gap.
- `lyric-line.tsx` — centered text, `break-words` / `overflow-wrap: anywhere`, responsive font clamps, `origin-center`, `contain`, `scroll-my-6`, compact-stage media query.
- `lyric-line-visual.ts` — reduced desktop active scale (1.04); no scale on mobile (`1.0`).
- `player-page.tsx` — `min-w-0 overflow-hidden` on lyrics column wrappers.
- `now-playing-header.tsx` — mobile-first title truncation.
- `transport-controls.tsx` — `min-w-0` on control row for narrow wrap.
