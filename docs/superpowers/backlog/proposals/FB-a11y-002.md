## Problem
Visually impaired users cannot follow which lyric line is currently active.

## Proposal
Add `aria-live="polite"` region that announces the active line text when it changes. Respect reduced-motion preference for announcement frequency.

## Acceptance criteria
- [ ] Active line announced on change
- [ ] Region is visually hidden (sr-only)
- [ ] Does not announce every word in word-level mode

## Constraints check
- [x] Client-only
- [x] No backend
- [x] Fits Vite SPA + Cloudflare static deploy
