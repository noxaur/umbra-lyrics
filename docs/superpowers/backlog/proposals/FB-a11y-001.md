## Problem
Users with low vision need larger lyrics than the default clamp() sizing provides.

## Proposal
Add an accessible range slider (aria-label "Lyric font size") scaling from 80% to 200%. Persist in localStorage.

## Acceptance criteria
- [ ] Slider in settings with visible value
- [ ] Lyrics scale smoothly without layout break
- [ ] Works with screen reader announcements

## Constraints check
- [x] Client-only
- [x] No backend
- [x] Fits Vite SPA + Cloudflare static deploy
