## Problem
Default accent colors may not meet WCAG AA contrast for users with color vision deficiency.

## Proposal
Third theme option "High contrast" with pure white active text on black stage, yellow highlight, no gradients.

## Acceptance criteria
- [ ] High contrast option in theme menu
- [ ] Active line contrast ratio ≥7:1
- [ ] Persists via localStorage

## Constraints check
- [x] Client-only
- [x] No backend
- [x] Fits Vite SPA + Cloudflare static deploy
